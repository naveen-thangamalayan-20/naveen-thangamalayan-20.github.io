AR File explorer

Goal

Create an interface that enchances the experiences for managing files remotely in Augmented reality environment

Process

We started with creating a mockup of the how the files and folders will look in augmeted reality environment. We considered using mouse as an input for interacting with the interface. We designed the prototype and ran a usability testing with couple of people and we identifed that it was difficult for users to access the interface using mouse. To improve the experience we used Leap Motion controller to detect the gestures of users. A pointer will be displayed when user moves his finger and to pick or do some operations a file he should make his hand a fist. This improved the experience for the users who tested the application.

Tools Used

Unity3D, NyArToolKit, Leap motion Controller

Math game

Goal

Create a casual game that is based on problem solving in Maths

Process

The objective of the game is that the user should solve the simple arithmetic problem in the given time. Iteration1:

We initially started with the the concept of showing an answer and user was given a option to choose what could be the correct arithmetic operation that could provide the result. But when we tested this with user, the user found it difficult to read the arithmetic operation.

Iteration2:

We then made it in such a way that numbers and arithmetic operators where displayed as buttons in bottom and users can select the button to derive the answer. If the number of options to the question where more user became frustrated and lost interest in the game. If the number of options where else it became too easy and users lost the fun of playing the game.

Iteration3:

We introduced only 4 options two of which were numbers and two are arithmetic operators the users need to derive the answer using this combination. This made it easy for the users to pick the options and also challenging because based on the operator choosed the answer might vary and they need to decide on the short time.

Tools Used

React Native, Figma
